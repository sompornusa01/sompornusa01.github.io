📖 วิธีอัปโหลดแบบ Manual
1. รัน python step.py ในเครื่อง
2. ไฟล์ step_result.json จะถูกสร้างใน docs/
3. commit และ push ขึ้น GitHub
4. Cloudflare Pages จะ auto-deploy ภายใน 1–3 นาที

4. โอน GitHub Repository และ Cloudflare Pages
